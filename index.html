A Few Accomplishments:

Consists of undergrad research and personal projects. Head over to my Github page to see more projects!

NYITFace

Implemented a face recognition app, that is supposed to replace the school ID card, during my undergrad research program. My teammates and I compared the eigen algorithm and fishman algorithm for training faces. We also created a UI to log in with a personal school ID number. The app would return an animated school logo if you were a verified attending student and if not a verified student, return a picture of an angry face.

Patient Manager

Coded a Java program in my early school years for hospital use. The patient manager was a simple UI connected to a SQL database. Functions include: adding new patient info, updating old patient info, deleting patient info, and showing initial and last visit.

Budgetting application using ReactJS/Redux

During an effort to learn more ReactJS and Redux, I created a budgetting application that allows you to enter a set budget and then shows you how much of your budget is left aftering entering whatever you have spent. I also created a log-in/sign-up page; however, the back end is not connected so I used the react router to route you to the home page after clicking sign in. Also, I wanted to create a dynamic pie chart that would categories what you have spent to visually show what your are spending most of your money on. I have yet to figure out how to update the pie chart using chartjs so, there is just a static pie chart on the page.
